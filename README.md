# Fintech Super App Frontend (Kotlin + Jetpack Compose)

This project is a complete frontend implementation of a Fintech Super App.

## Features
- Banking: AEPS, DMT, Mini ATM, Balance Enquiry, History.
- Payments: QR Scan, UPI, Wallet, Merchant Payments.
- Bill Payments: Mobile, DTH, Electricity, Water, Gas.
- Travel: Bus, Train, Flight, Hotel Booking.
- Govt Services: PAN Card, GST, ITR filing.
- Job Listing & Support.

## Architecture
- MVVM + Unidirectional Data Flow.
- Jetpack Compose (Material 3).
- Hilt for Dependency Injection.
- Jetpack Navigation with Nested Graphs.
